# Grand Unifying Theory Potential Framework

**A Unified Potential for the Grand Unification of Fundamental Forces**

MATLAB implementation of the variational efficiency thesis (Christopher Ford Morgan, 2026).

This repository contains a complete, self-contained numerical simulator for a new Grand Unifying Theory (GUT) based on a single variational principle. The framework derives gravity, electromagnetism, weak and strong forces, dark energy, and resolves several cosmological tensions (Hubble, S8, lithium problem, voids, etc.) without a cosmological constant.

## Features
- Single unified potential V(E) that unifies all fundamental forces
- Dynamic SU(5) → SU(3)×SU(2)×U(1) symmetry breaking with Mexican Hat potential
- Cosmic time evolution of the scalar field using ODE solver
- Proton decay lifetime and branching ratios (smoking-gun predictions)
- Right-handed neutrino dark matter candidate via seesaw mechanism
- Bayesian model comparison vs ΛCDM
- Internal cross-validation of all resultants
- One-click publication-ready PDF report generation
- Full QFT consistency (one-loop Coleman-Weinberg corrections + anomaly check)

## How to Run
1. Download or clone this repository.
2. Open MATLAB.
3. Run the main script:
   ```matlab
   run gut_framework.m
